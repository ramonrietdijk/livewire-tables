# Requirements

Before proceeding, please make sure you have **at least** met the following prerequisites:

::: info
Note that Alpine.js is included in Livewire 3 by default.
:::

| Requirement  | Version       |
|--------------|---------------|
| PHP          | `^8.1`        |
| Laravel      | `^9.0, ^10.0` |
| Livewire     | `^2.0, ^3.0`  |
| Alpine.js    | `^3.0`        |
| Tailwind CSS | `^3.0`        |
